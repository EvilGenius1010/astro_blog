# HK's Blog

This is a blog site built by **Astro** deployed on **Cloudflare Pages**.
To try it out, bring it locally by cloning it using

```
git clone https://github.com/EvilGenius1010/astro_blog.git.
```

Once done, run

```
npm install
npm run dev
```

to install dependencies and start the server locally.

You can now see the files being served at **http://localhost:4321/** .
